# Shades Colour Theme

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT) ![Maintenance](https://img.shields.io/maintenance/yes/2021) ![GitHub contributors](https://img.shields.io/github/contributors/ansh-sarkar/solvable-shades) ![GitHub last commit](https://img.shields.io/github/last-commit/ansh-sarkar/solvable-shades)

Shades is a colour theme which was made keeping in mind the high contrast which programmers often demand in order to be able to perform faster coding , greater legibility and to impart a somewhat soothing effect when exposed to long hours of screen time .
